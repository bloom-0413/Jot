## 关于

Jot（中文名：简记）是一款笔记App  
适用于 Android12 及以上的系统

## 特性

1. 无广告、无追踪、无内购
2. 使用 Material Design 3 设计
3. 体积轻量（仅2.5MB）
4. 支持本地备份
5. 支持在线更新
6. 支持归档笔记

## 下载

请前往 [Release 页面](https://github.com/bloom-0413/Jot/releases) 下载最新版本
