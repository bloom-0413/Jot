## 🔍 关于

Jot（中文名：简记）是一款笔记 App

基于 Kotlin + Jetpack Compose 构建  
适用于 Android 12 及以上的系统

## 🛠️ 特性

1. 无广告、无追踪、无内购
2. 使用 Material Design 3 设计
3. 恰到好处的动画
4. 体积轻量（仅 2.5 MB）
5. 支持本地备份
6. 支持在线更新
7. 支持归档笔记
